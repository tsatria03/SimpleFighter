These are the available settings you can change when building character blockers.

building: Allows or blocks adding and editing objects in the map using B and U.
camera: Allows or blocks full camera usage with G. This includes all camera controls such as R, Y, J, K, L, I, M, and T.
commanding: Allows or blocks usage of the slash command console.
drawing: Allows or blocks drawing shields and weapons using S and W.
firing: Allows or blocks firing weapons with Control and reloading with R.
healing: Allows or blocks healing your character and shields using Right Bracket and Backslash.
jumpheight: Allows or blocks adjusting jump height using F1, F2, and F3.
jumping: Allows or blocks jumping with space.
movespeed: Allows or blocks changing movement speed using A, D, and F.
quitting: Allows or blocks quitting or leaving the current map with escape.
running: Allows or blocks running by holding Alt.
scriptkeys: Allows or blocks using script keys with the number row.
spying: Allows or blocks using the spire with Shift + arrow keys.
teleporting: Allows or blocks random teleportation using J, K, and L.
turning: Allows or blocks character turning using Alt + Shift + arrow keys.
using: Allows or blocks using items and shields with Shift + Enter and O.

Character blockers can also operate in blacklist and whitelist modes.
Blacklist (false): Only the listed settings are blocked. All other settings remain enabled.
Whitelist (true): Only the listed settings are allowed. All other settings are blocked.
